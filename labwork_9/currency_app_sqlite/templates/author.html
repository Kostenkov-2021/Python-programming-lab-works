О проекте

Currency Tracker - это веб-приложение для отслеживания курсов валют в реальном времени. Приложение разработано в рамках лабораторной работы по дисциплине "Программирование на Python".

Технологии

Python HTTP Server Jinja2 Chart.js HTML/CSS JavaScript

Функциональность

  • Получение актуальных курсов валют от ЦБ РФ
  • Управление пользователями и их подписками
  • Визуализация динамики курсов с помощью графиков
  • API для интеграции с другими системами
  • Адаптивный веб-интерфейс

class User:
    def __init__(self, user_id: int, name: str):
        self._id = user_id
        self._name = name
        self._subscriptions = []
    
    @property
    def id(self) -> int:
        return self._id
    
    @id.setter
    def id(self, value: int):
        if not isinstance(value, int):
            raise TypeError("ID должен быть целым числом")
        if value <= 0:
            raise ValueError("ID должен быть положительным")
        self._id = value
    
    def subscribe_to_currency(self, currency_id: str):
        if not any(sub.currency_id == currency_id 
                   for sub in self._subscriptions):
            subscription = UserCurrency(
                len(self._subscriptions) + 1,
                self.id,
                currency_id
            )
            self._subscriptions.append(subscription)
